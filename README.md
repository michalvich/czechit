# Czechit

The purpose of this gem is provide validations of phones, identification numbers, VAT identification numbers for Czech Republic

## Installation

Add this line to your application's Gemfile:

    gem 'czechit'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install czechit
